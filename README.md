# 🛒 Retail Sales Analysis 2023

## 📌 Project Overview
Analisis data penjualan retail menggunakan SQL, Excel, dan Power BI
untuk mengidentifikasi tren penjualan, performa kategori produk,
dan segmentasi pelanggan.

## 🛠️ Tools Used
- **PostgreSQL** — Data cleaning & querying
- **Microsoft Excel** — Data analysis & visualization
- **Power BI** — Interactive dashboard

## ❓ Business Questions
1. Kategori produk mana yang menghasilkan revenue tertinggi?
2. Bulan mana yang memiliki penjualan tertinggi?
3. Bagaimana perbandingan revenue antara pelanggan Male vs Female?
4. Kelompok usia mana yang paling banyak bertransaksi?

## 💡 Key Insights
- Electronics menjadi kategori dengan revenue tertinggi (156.905)
- Mei adalah bulan dengan penjualan terbaik (53.150)
- Male sedikit lebih tinggi (51%) dibanding Female (49%)
- Tidak ada repeat customer → bisnis perlu strategi retensi pelanggan
